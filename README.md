# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit that helps users analyze resumes and get suggestions for improvement.

## Features

- Upload resume files
- Extract important information from resumes
- Analyze skills and keywords
- Provide resume improvement suggestions
- Simple and user-friendly interface
- Fast and lightweight web application

## Technologies Used

- Python
- Streamlit
- Machine Learning
- Natural Language Processing (NLP)
- Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/AI-Resume_Analyzer.git
```

2. Go to the project folder

```bash
cd AI-Resume_Analyzer
```

3. Install required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```

## How It Works

1. Upload your resume.
2. The system extracts text from the resume.
3. AI analyzes the content.
4. The application provides:
   - Skills identified
   - Missing keywords
   - Resume improvement suggestions
   - Overall analysis
